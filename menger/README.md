# Menger

This project implements the Menger Sponge fractal in C.

## Description

The Menger Sponge is a three-dimensional fractal curve. This project generates a 2D representation of the Menger Sponge at a given level.

## Usage

Compile the program:

```bash
gcc menger.c -o menger
```

Run the program with the desired level:

```bash
./menger <level>
```
